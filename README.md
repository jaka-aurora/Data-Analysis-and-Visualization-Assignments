## Data Analysis and Visualization

This repository contains assignments completed for the Data Analysis and Visualization course at JAMK.  
It demonstrates techniques in **data exploration, cleaning, transformation, visualisation, and communication of insights** using Python and Jupyter Notebooks.

## Skills Demonstrated

- Data loading and preprocessing  
- Exploratory Data Analysis (EDA)  
- Creating effective visualisations  
- Drawing insights from data  
- Communicating results through annotated notebooks

## How to Run

1. Clone this repository.
2. Create and activate a virtual environment
    ```
    python3 -m venv venv
    ```
3. Install dependencies:
   ```
    pip install -r requirements.txt
   ```
4. Launch Jupyter Notebook.
5. Open any assignment notebook in the _assignments_ folder and run the cells.  

> Each assignment focuses on a different aspect of data analysis and visualisation, such as:  
>   - Loading and cleaning datasets  
>   - Exploring data distributions  
>   - Visualising data with charts and plots  
>   - Interpreting patterns and insights
> 
> These notebooks show the progression of analytical thinking and technical skills relevant for data-focused roles.
